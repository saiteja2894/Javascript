Javascript Cheat Sheet

JavaScript

Javascript used to incorporate actions to webpages

JavaScript Types

Numbers -- Same as other languages. we can use +, - , % , /, * operations on numbers. Number() to convert string to numbers

String -- you can use either single quotes or double quotes. '+' is used for appending strings and you can use backslash(\) to escape identifiers.

Boolean -- true and false.

undefined -- variable is not assigned

JavaScript Comparisons

=== is used to compare two types are same. !== is for unequal and all the other comparison operators are same as other languages <=, >=, <, >. Comparison operators return boolean type.

JavaScript Variables

var -- this keyword is used to define a variable in js. variable names must start with letter but can end with number. It cannot start with symbols other than _ and $.

prompt(string) -- It is like scanf in c. Gets the input from the user. String argument passed into the function is for displaying the message when prompt is asking the user to input.

alert(string) -- This is like printf.

semicolon (;) is used to represent the end of the statement.

JavaScript Conditionals

javascript supports if, else, else if as many other languages
Here is an example of the usage
if (condition1) {
statement1;
} else if (condition2) {
statement2;
} else {
statement3;
}
you can use "||", "&&", "!" operators in the condition
